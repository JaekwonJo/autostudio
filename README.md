# autostudio

자동화 스튜디오 HTML 도구 보관 저장소입니다.

## 포함 도구
- `⚡자동화스튜디오.html`
  - 파일관리에서 일반 `S###` 표준화 변경과 별도로 `S### -> @S###` 전용 일괄 변경 버튼 지원
- `유튜브_쇼츠_API_발굴기.html`
  - 화면 이름: `유튜브 핫토픽 레이더`
  - 공식 YouTube Data API v3 기반
  - 고조회수 Shorts 후보 검색 + 오늘의 고조회수 영상 + 최근 N일 고조회수 영상 검색 지원
  - 경제 / 역사 / 교육 / 과학 / AI / 시사 / 자기계발 주제 프리셋 지원
  - 검색 후 조회수 / 길이 / 날짜로 다시 필터링
