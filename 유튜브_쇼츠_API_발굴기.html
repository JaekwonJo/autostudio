유튜브 핫토픽 레이더

크롤링이 아니라 공식 YouTube Data API v3로 찾는 도구입니다. 이제 쇼츠 발굴뿐 아니라 오늘 올라온 영상 중 조회수 높은 순, 최근 N일 기준 고조회수 영상도 같이 볼 수 있게 만들었습니다. 경제 / 역사 / 교육 같은 주제 프리셋도 함께 지원합니다.

공식 시작 문서 공식 쿼터 문서 search.list 100 units videos.list 1 unit

설정

🔐 API 키 설정 펼치기
이 HTML은 API 키를 브라우저 localStorage에만 저장합니다.
주의: file:// 로 여는 로컬 HTML은 HTTP referrer 제한 키가 잘 안 맞을 수 있습니다. 개인용이면 API 제한만 YouTube Data API로 걸고 테스트하는 쪽이 가장 단순합니다.
주제 프리셋을 고르면 아래 검색어와 자동으로 합쳐집니다. 검색어를 비우면 너무 넓게 잡혀서 품질이 떨어질 수 있습니다.
현재는 직접 입력 모드입니다.
예: 100000, 200000, 500000, 1000000
쇼츠 모드에서는 videoDuration=short에 더해 여기서 한 번 더 좁힙니다.
0이면 날짜 제한 없이 검색합니다.
1페이지는 최대 50개입니다.
쿼터 계산 예시
검색 1페이지 = search.list 100 units
상세 조회 50개 = videos.list 1 unit
즉 보통 2페이지 검색 + 상세 100개 조회면 대략 202 units 정도라서, 공식 기본 일일 한도 10,000 units 안에서 개인용은 충분한 편입니다.

이번 실행 요약

API 키를 넣고 검색어를 적은 다음 고조회수 쇼츠 찾기를 누르세요.

사용 팁

1. `오늘의 고조회수 영상`은 보통 정렬: 조회수, 최근 며칠: 1처럼 쓰면 좋습니다.
2. 경제 / 역사 / 교육처럼 프리셋을 먼저 고르고, 필요하면 검색어를 더 붙여 좁혀보세요.
3. 최소 조회수를 10만, 20만, 50만, 100만 식으로 바꿔보세요.
4. 쇼츠 모드는 짧은 영상 + 60초 이하 기준의 실전용 근사치입니다.
